自主制作

一 成長は、プロセスに興味があるところから始まる 。

RGB HEX Color Code Visualizer

Интерактивный веб-инструмент, предназначенный для визуализации процесса формирования цвета через шестнадцатеричные (HEX) коды. Проект позволяет в реальном времени наблюдать, как изменение значений в каналах Red, Green и Blue влияет на итоговый цвет.

JS Array Methods Visualizer

Образовательный веб-инструмент, предназначенный для наглядного изучения работы встроенных методов массивов JavaScript.
Проект позволяет интерактивно манипулировать данными и мгновенно видеть результат выполнения кода, что помогает лучше понять внутреннюю логику работы методов.

Japanese Business Email Builder

Интерактивный веб-инструмент для автоматизации составления деловых писем на японском языке в соответствии с нормами бизнес-этикета. Проект позволяет пользователям динамически формировать структуру письма, настраивая заголовки, приветствия, основную часть и подпись в реальном времени.

Dynamic Twitch Chat Overlay

Высокопроизводительный веб-виджет, предназначенный для отображения сообщений из чата Twitch в реальном времени во время прямых трансляций.
Инструмент интегрируется непосредственно с Twitch IRC и сторонними сервисами через API, обеспечивая плавную визуализацию общения зрителей.

Car Care Service Configurator

Интеллектуальная система конфигурации услуг для автомобилей. Проект прошел путь от авторского концепта до внедрения в production.
В основе сервиса лежит уникальная логика взаимозависимых параметров, которая впоследствии была защищена патентом.
