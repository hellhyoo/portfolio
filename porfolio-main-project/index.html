自主制作

一 成長は、プロセスに興味があるところから始まる 。

RGB HEX Color Code Visualizer

HEX コードを通じて色の形成プロセスを視覚化するインタラクティブなウェブツール。 このプロジェクトでは、Red、Green、Blue の各チャンネルの値の変化が最終的な色にどのように影響するかをリアルタイムで観察できます。

JS Array Methods Visualizer

JavaScript の組み込み配列メソッドの動作を視覚的に学習するための教育用ウェブツール。 このプロジェクトでは、データをインタラクティブに操作し、コードの実行結果を即座に確認できるため、メソッドの内部ロジックをより深く理解することができます。

Japanese Business Email Builder

ビジネスマナーに準拠した日本語のビジネスメール作成を自動化するインタラクティブなウェブツール。 このプロジェクトでは、ユーザーがメールの構造を動的に構成し、見出し、挨拶文、本文、署名をリアルタイムで調整できます。

Dynamic Twitch Chat Overlay

ライブ配信中に Twitch チャットのメッセージをリアルタイムで表示する高性能ウェブウィジェット。 このツールは Twitch IRC および API を通じてサードパーティサービスと直接統合し、視聴者のコミュニケーションをスムーズに視覚化します。

Car Care Service Configurator

自動車向けサービス設定のインテリジェントシステム。このプロジェクトは独自のコンセプトから本番環境への実装まで進化しました。 サービスの核心には相互依存パラメータの独自ロジックがあり、後に特許で保護されました。
